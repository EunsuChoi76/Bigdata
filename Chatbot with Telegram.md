
## 용어 준비물
BotFather : bot 생성관리 계정

## 단말상 명령어
/start
/newbot

## 진행단계
 단말에 텔레그램 app 설치
 BotFather 검색추가
 /start 로 시작
 /newbot
 bot이름 입력
 username 입력
 token 발급되면 저장해 놓음

 파이썬모듈 설치
   1) pip 이용시
       $pip install python-telegram-bot --upgrade
   
   2) 소스코드로 직접 설치시
       $git clone https://github.com/python-telegram-bot/python-telegram-bot
       $cd python-telegram-bot
       $python setup.py install
